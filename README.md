## 📝 스터디 규칙
 - Pull Request 방식을 사용합니다.
 - branch는 master만 사용합니다.
 - 매주 각자 공부가 필요한 주제를 선정 후 학습합니다.
 - 모든 문서는 마크다운 형식을 이용하여 작성합니다.
 - Repo에 공유는 일요일까지 완료합니다.
 - 매주 화요일 소공 수업 후 대면수업 시 오프라인, 비대면 시 디스코드로 발표 진행합니다.
